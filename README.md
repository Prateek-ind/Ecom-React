# 🛒 HealthyBuddie Clone (React)

A frontend clone of the HealthyBuddie website built using React. This project focuses on replicating the core UI and functionality of a modern e-commerce platform.

---

## 🚀 Features

- 🏠 Home page with product listings
- 🔍 Product browsing and search functionality
- 📄 Product detail view
- 🛒 Add to cart functionality
- ➕ Update cart quantity
- ❌ Remove items from cart
- 🛒 Checkout functionality
- 🛒 Bulk order inquiry
- ⚡ Responsive UI

---

## 🛠️ Tech Stack

- React.js
- JavaScript
- Redux Toolkit
- React Router
- Tailwind 

---

## 📂 Folder Structure

src/
├── assets/ # Images, icons, static files
├── features/ # Feature-based modules (e.g., cart, products)
├── hooks/ # Custom React hooks
├── layouts/ # Layout components (Navbar, Footer, etc.)
├── pages/ # Route-based pages
├── services/
│ └── firebase/ # Firebase configuration & services
├── shared/
│ └── components/ # Reusable UI components
├── store/ # Redux store & slices
├── utils/ # Helper functions
